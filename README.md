# Nginx

A high performance web server and a reverse proxy server.

## Docker

This Docker image is using [Ubuntu 22.04](https://hub.docker.com/_/ubuntu/) as base image.

## Ports

| Port    | Description |
| ------- | ----------- |
| 80/tcp  | HTTP        |
| 443/tcp | HTTPS       |
